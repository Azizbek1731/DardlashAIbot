# Mental Health Support Telegram Bot

🤖 AI-powered mental health support bot for Telegram

## Features
- Multi-language support (Russian, English, Uzbek)
- Crisis detection and intervention
- Voice message support
- Structured therapeutic conversations
- Beta testing: Limited to 50 users

## Tech Stack
- Python 3.11+
- python-telegram-bot
- OpenAI GPT-4
- Muxlisa API (Uzbek TTS/STT)

## Deployment
Deployed on Render.com with automatic deploys from GitHub.

## Configuration
All sensitive configuration is managed via environment variables (not in repo).

## Beta Testing
Current status: Private beta (invite-only)
